# Product Listing App

CodeIgniter 4 + Vue.js 3 Composition API Product listing application.

## Tech Stack

**Backend:**
- CodeIgniter 4
- MySQL

**Frontend:**
- Vue.js 3 (Composition API)
- Axios

## Setup

### Backend
```bash
cd product-api
composer install
```

### Frontend
```bash
cd product-client
npm install
npm run dev
```

## Done!
